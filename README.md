# My Car Gallery

An interactive car gallery web app with features like drag & drop upload, image preview, favourites, downloads, and 3-dot menu actions. Built with **HTML**, **CSS**, and **JavaScript**.

---

## Features

- **Preloaded Gallery:** Comes with 12 preloaded car images.  
- **Drag & Drop Upload:** Upload images by dragging or browsing. Preview images before adding to gallery.  
- **Add / Reset Uploads:** Add selected images to gallery or reset the upload preview.  
- **3-Dot Menu on Cards:** Each image has a menu with options:  
  - Add to Favourites  
  - Download / Download Again  
  - Delete / Restore  
  - Edit Caption  
- **Section Management:**  
  - Dashboard (Your Images)  
  - Favourites  
  - Downloads  
  - Deleted / Trash  
- **Bulk Actions:**  
  - Remove All from Favourites  
  - Remove All from Downloads  
  - Clear Bin / Restore All in Deleted section  
- **Responsive Design:** Works on desktop and mobile screens.  
- **Empty Section Indicator:** Shows message when a section has no images.
